# translation philosophy
- accurate meaning over poetic form (nicholson-style faithful prose)
- preserve sufi register, dignified plain shahmukhi punjabi
- no forced rhyme, no meter constraints
- allow 1:1, 1:2, or 2:1 couplet alignment when meaning needs breathing room

# script rules
- output exclusively in shahmukhi (perso-arabic) script
- never gurmukhi, never devanagari, never roman transliteration in final output
- shahmukhi uses the same perso-arabic alphabet as farsi/urdu with some additional letters
